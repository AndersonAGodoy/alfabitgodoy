# **Design System da Alfabit Godoy** 🎨

### **Bem-vindo ao Design System da Alfabit!**

Este Design System foi criado para trazer consistência, acessibilidade e eficiência no desenvolvimento de interfaces para os produtos da Alfabit Godoy.

Desenvolvido com **Angular 17**, o sistema é projetado para oferecer uma base sólida de componentes reutilizáveis e configuráveis, facilitando a criação de experiências coesas e personalizadas.

---

## **Objetivo**

O objetivo principal deste Design System é:
- **Centralizar decisões de design** para manter a identidade visual.
- **Aumentar a produtividade** dos desenvolvedores ao oferecer componentes prontos para uso.
- **Promover a acessibilidade**, tornando as aplicações inclusivas para todos.

---

## **Estado do projeto**

Atualmente, o projeto está em desenvolvimento, com os primeiros componentes já disponíveis para uso:
- **Button** – Um botão altamente personalizável e acessível.
- **Input** – Um campo de entrada com suporte a estados e validações.

Estamos continuamente expandindo o conjunto de componentes para atender às necessidades da equipe e dos produtos da Alfabit.
