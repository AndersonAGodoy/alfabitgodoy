# **Como usar o componente Button**

O componente **Button** foi projetado para ser simples, reutilizável e personalizável. Veja abaixo como integrá-lo à sua aplicação Angular.

---

## **Importando o Button**

Para utilizar o componente em seu projeto Angular, basta importar o módulo correspondente:

```ts
import { ButtonComponent } from '@alfabitgodoy/button';
```
Após a importação, basta adicionar o botão em seu template HTML desta forma:
